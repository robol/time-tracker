Welcome to TimeTracker website

TimeTracker is a simple software that lets you keep track of the time spent on a given task by summing duration of events in your Google Calendar. You just let it know the string that it should look for, and the interval that you'd like to monitor.

The software is still work in progress, so don't expect it to work flawlessly.

Building TimeTracker

To build TimeTracker you'll need Qt4, libqjson and CMake. I think that Qt5 will work as well, and you shouldn't even need libqjson in that case. 

$ git clone git://github.com/robol/time-tracker
$ cd time-tracker
$ mkdir build && cd build/
$ cmake ..
$ make
